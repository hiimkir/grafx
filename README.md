# Обзорщик 3Д моделей

## Запуск:

В терминале указывается путь к 3D модели формата .OBJ. Если путь не указан, то берётся модель куба из папки `/assets`

## Управление:

<img width="802" height="627" alt="image" src="https://github.com/user-attachments/assets/b6d8c450-cd27-49b0-ba17-ee4957974501" />

- Модель можно вращать зажатым курсором
- Камеру можно приближать стрелками вниз и вверх
- Клавишей `c` можно включать/выключать прозрачность граней
- Направление освещения можно вращать стрелками вправо и влево
